Sorting Visualizer

About


Sorting is nothing but arranging a string of numbers or other elements in the correct order. This project Sorting Visualizer is a UI which allows the users to select the array size,speed of the visualization and select the sort algorithm.
The problem Sorting Visualizer solves. Created a web application using HTML, CSS, Javascript to visualize how various sorting algorithms work.
